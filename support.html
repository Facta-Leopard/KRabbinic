KSaju Support

Use this page for KSaju support, bug reports, feature requests, and app-scope questions.

English

Before You Report an Issue

Please check the following:

  • KSaju requires iOS 26 or later.
  • Saju Reading is implemented in the initial release.
  • Today's Fortune, Compatibility, and Twelve Animal Signs are reserved for future updates.
  • Apple Intelligence interpretation may fall back to the built-in on-device reading when Apple Intelligence is unavailable, disabled, not ready, or unsupported for the selected language.

What to Include

When reporting an issue, include:

  • Device model.
  • iOS version.
  • KSaju app version.
  • App language setting.
  • Birth profile input pattern, if relevant. Do not share sensitive personal birth data unless you intentionally choose to.
  • Steps to reproduce the issue.
  • Screenshots, if they help explain the problem.

Privacy and Sensitive Data

Do not post private personal information in a public GitHub issue. If a birth profile is needed to reproduce a calculation issue, you may use an approximate or fictional test profile unless exact details are necessary.

App Scope

KSaju readings are for self-reflection and entertainment. KSaju does not provide professional health, legal, financial, medical, immigration, employment, or safety advice.

Where to Request Help

Open a GitHub issue in the KSaju project repository and choose the most relevant label, such as bug, question, or feature request.

한국어

문의 전 확인 사항

먼저 다음 사항을 확인해 주세요.

  • KSaju는 iOS 26 이상에서 사용할 수 있습니다.
  • 초기 버전에서는 사주풀이 기능이 구현되어 있습니다.
  • 오늘의 운세, 궁합, 십이간지는 향후 업데이트 예정 기능입니다.
  • Apple Intelligence 해설은 Apple Intelligence를 사용할 수 없거나, 꺼져 있거나, 아직 준비되지 않았거나, 선택 언어를 지원하지 않는 경우 기기 내장 해설로 대체될 수 있습니다.

제보 시 포함할 내용

문제를 제보할 때는 다음 정보를 포함해 주세요.

  • 기기 모델
  • iOS 버전
  • KSaju 앱 버전
  • 앱 언어 설정
  • 관련이 있는 경우 출생 프로필 입력 패턴. 민감한 실제 생년월일 정보는 사용자가 명시적으로 원하지 않는 한 공개하지 마세요.
  • 문제 재현 단계
  • 문제 설명에 도움이 되는 스크린샷

개인정보와 민감 정보

공개 GitHub 이슈에는 민감한 개인정보를 올리지 마세요. 계산 문제 재현에 출생 프로필이 필요한 경우, 정확한 정보가 꼭 필요한 상황이 아니라면 근사값이나 가상의 테스트 프로필을 사용할 수 있습니다.

앱 제공 범위

KSaju의 사주 해설은 자기이해와 엔터테인먼트 목적의 참고 정보입니다. KSaju는 건강, 법률, 금융, 의료, 이민, 고용, 안전 관련 전문 조언을 제공하지 않습니다.

지원 요청 위치

KSaju 프로젝트 GitHub 저장소에서 이슈를 열고, bug, question, feature request 등 가장 알맞은 라벨을 선택해 주세요.
